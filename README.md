# Illegal Immigration and Drug Trafficking Monitoring Dashboard

## Overview
A web-based dashboard developed using HTML, CSS, and JavaScript to monitor incidents related to illegal immigration, drug cases, and drug trafficking.

## Features
- Add Incident Records
- Search by Location
- Delete Records
- Immigration Monitoring
- Drug Case Tracking
- Drug Trafficking Tracking

## Technologies Used
- HTML5
- CSS3
- JavaScript
